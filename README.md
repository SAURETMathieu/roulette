# next-template

A Next.js 14 template for building apps with Radix UI and Tailwind CSS.

## Features

- Next.js 14 Src Directory
- Radix UI Primitives
- Tailwind CSS
- Icons from Lucide
- Dark mode with `next-themes`
- Tailwind CSS class sorting, merging and linting.
- Prettier and ESLint
- i18n with `next-intl`
- Notifications with `sonner`
- 2 navigation models

## Installation

```bash
npm install
```

## Launch the App

```bash
npm run dev
```
